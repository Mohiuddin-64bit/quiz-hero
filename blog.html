Today's Question

Question: What is the difference between map(), filter(), find(), and forEach()? Sent message.

Answer: map() and filter() both return new arrays, while find() and forEach() do not. map() is used to transform an array, while filter() is used to select elements from an array. find() is used to find a single element in an array, while forEach() is used to execute a function on each element of an array.

Question: What is local storage and session storage and cookies in JS?

Answer: localStorage and sessionStorage are both JavaScript APIs that allow you to store data on the browser.localStorage stores data with no expiration date, while sessionStorage stores data for only one session.
cookies are small text files that are stored on the client-side by the browser. They are used to store data that can be accessed by the server. Cookies can be set to expire after a certain amount of time, or they can be set to expire when the browser is closed.

Question: what is event loop, how does it work?

Answer: The event loop is a fundamental concept in JavaScript that allows it to handle asynchronous operations. It is a constantly running process that monitors both the call stack and the callback queue. When the call stack is empty, the event loop takes the next function from the callback queue and pushes it onto the call stack to be executed.

Question: What is the quizTimer function in JavaScript, and how is it used to create a timer for a quiz?

Answer: The quizTimer function is a JavaScript function that takes a single argument dismiss. The function is used to create a timer for a quiz. If the dismiss argument is true, the function clears the timer and returns. Otherwise, the function sets an interval that runs every second. The interval updates the timer display on the page and changes the color of the timer every 60 seconds.

Question: What is the chooseQuiz function in JavaScript, and how is it used to add quiz answers to an array?

Answer: The chooseQuiz function is a JavaScript function that takes two arguments index and givenAns. The function is used to choose a quiz answer and add it to an array of answers. If the answer already exists in the array, the function updates the existing answer. Otherwise, the function adds the new answer to the array.
